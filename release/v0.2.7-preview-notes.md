# CS2FOW v0.2.7-preview

This release candidate supports CS2 build `24304127` and the pinned Metamod:Source/HL2SDK interface.

## What changed

- Replaced the fifteen hand-tuned body points and eight AABB corners with Valve's nineteen live animated hitbox capsules.
- Visibility now evaluates the complete projected capsule volumes through a bounded CPU depth buffer. Any visible capsule sample or the separate weapon-muzzle point reveals the whole player.
- Preserved the existing viewing origins, smoke occlusion, HE smoke clearing, visibility hold, teammate rules, and CheckTransmit behavior.
- Invalid or incomplete capsule capture, uncertain sub-pixel coverage, worker cancellation, or exhaustion of the 75 ms worker budget fails open and leaves players visible.
- Added capsule capture and solver counters to `cs2fow_status`; the local Studio keeps its legacy point solver for comparison.

## Installing the test build

1. Update the CS2 server and Metamod:Source.
2. Extract the Windows core ZIP into the server's `game/csgo` folder without rearranging it.
3. Load a map and run `cs2fow_status`.

CS2FOW intentionally remains inactive if the loaded CS2 server binary does not match the verified build fingerprint.

## Validation

The Windows build, runtime tests, package tests, Studio consistency checks, and the 20-player active-smoke benchmark pass. The final Steam Runtime Linux artifact and live-server validation remain pending before publication.
